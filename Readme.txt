コンパイル方法とプログラムの動かし方を説明する。以下をコピーアンドペーストで実行できる。

＜RBN.cについて＞
コンパイルは以下をコピーすれば良い。

$ gcc -o RBN RBN.c -lm
$ ./RBN

これを実行すると、以下のような文が出る。

    ノードの数(4~7)を入力してください...

ここで4~7の数字を入力しEnterをおせばノード数を入力できる。
次に以下のような文が出る。

    ノードの次数(2~4)を入力してください...

ここで2~4の数字を入力しEnterをおせばノードの次数を入力できる。
次に以下のような文が出る。

    コンスタント関数の有無(0:無し, 1:有り)を設定してください...

ここで0を入力しEnterを押せばコンスタント関数をなしに、1を入力すればコンスタント関数をありにできる。
次に以下のような文が出る。あとは表示される通りにEnterを押せば、結果が順に表示される。


＜PBN.cについて＞
コンパイルは以下をコピーすれば良い。

$ gcc -o PBN PBN.c -lm
$ ./PBN

これを実行して出る文は、コンスタント関数の有無までは同じなので省略する。
コンスタント関数を決定した後に、次の文が出る。

    関数の出力が1となる確率(%)を入力してください...

ここに0~100の値を入力し、Enterを押せば確率が入力できる。本実験では70である。
これ以降はRBNと同じであるので省略する。
